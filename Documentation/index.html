Installation #back to top

  • Step 1 - Unzip the Team Collaboration System And Project Management Tool.zip which you have downloaded from my github account.
  • Step 2 - Find the Code folder from the downloaded package and take out all the data from Code folder and upload to your server.
  • Step 3 - Open your web browser and go to your domain url. For example: http://yourdomain.com Now you will be redirected to the installation page.
  • Step 4 - Enter your the Database Details, if not created databse then create a MySql Database via your web hosting providers Control Panel. Then, enter your Admin Email and Password which will be required to login in the application. This credentials will needed on everytime you login.
  • NOTE : - Password requires 8 characters minimum and no whitespace.
  • Step 5 - Now click on install and wait untill process finished. The system will create necessary tables in database, files and configuration, Then you will be automatically redirected to the login page.
  • NOTE : - TeamWork is now ready to run but we recommended you to first setup and customized TeamWork application according to your needs from settings section.

General Settings #back to top

  • In this section, you can change the application logos, title, footer, file type restriction and many more.
  • NOTE : - Make sure you setup this section because it will comes up with TeamWorks default values.

Email Settings #back to top

  • This setting is for out-going emails. All emails will be sent using this setting so make sure add your email smtp setting here. Without email setting email functionality will not going work.

Update #back to top

  • One click update system. Here you can update your application. You just have to upload your update zip file and rest leave on our system.
  • NOTE : - Please backup all files and database before start the installation of updates.

Subscription Plans #back to top

  • SaaS admin can create, edit and delete subscription plans here.
  • NOTE : - Default subscription plan can't be deleted.

Users and Plan #back to top

  • SaaS admin can view details of user who have purchased subscription plans also can manage things like create, edit and delete users and thier subscription plans here.

SaaS Admins #back to top

  • You can manage SaaS admins or SaaS users here.

Subscription and Billing #back to top

  • From here user can purchase subscription plans.

Profile #back to top

  • From profile section you can update your profile details and also able to see some basic statistics of your account.
  • NOTE : - Email can not be updated once its added.

Projects #back to top

  • All you project will be listed here with some basic information. You can manage your projects here like create, update, delete, add users and filter projects.
  • When you click on one specific project you will get all the information about this project, this page known as project details page. Here also you can manage your project like create, update, delete and some extra features.

Tasks #back to top

  • All you tasks will be listed here with some basic information. You can manage your tasks here like create, update, delete assing them to any user. You and user can drag and drop task in the process queue. From projects section you can select tasks for specific project else all task will be listed here.

Users #back to top

  • All system user will be listed here with some basic profile statistics. Using this application you can manage your team members easily. Here you can Add, View, Edit, Delete, Active and Deactive users from the application.

Permissions #back to top

  • From here admin can handle everything about users permissions in application. Admin can assign any permission to the users.
  • NOTE : - Do not give settings permissions to the users.
